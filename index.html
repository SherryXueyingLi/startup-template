TabConfig

In config.js file, tabs defined the tab label, state name, and controller path, url is defaultly set. Literally, it could support infinity sub-tabs.

function appendContext help append elements based on angularJs. It could support two level's tabs. So any tab defined but lower then level two will be ignored.

Routing

By using 'resolve', it help load the controller and trigger a click event after successfully loaded.
